git仓库存储提交历史的元数据
.git    .开头的是隐藏目录/文件 默认不显示
用来存储用户不需要直接与之交互的数据
用ls -a 可以看见

————————————————————————————
git init //初始化仓库
git status//检查状态
初始化后没有任何提交

————————————————————————————
git add
git add .
git reset <文件名> //从暂存区删除

————————————————————————
git commit //提交，默认打开编辑器
git commit -m "Commit message" //简化版
commit message要用祈使句
